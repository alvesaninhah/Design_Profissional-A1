Sobre a Entrega

Este repositório contém os arquivos referentes à minha avaliação para o primeiro semestre do curso de Ciência da Computação.
Todos os materiais aqui apresentados foram cuidadosamente organizados e entregues para validar minha participação nas atividades e competências desenvolvidas até o momento.

Estrutura do Repositório

A estrutura do repositório é organizada da seguinte maneira:

Arquivos

Curriculo_Latex.pdf: Meu currículo atualizado, com a experiência e as qualificações adquiridas até o momento no curso.

Certificado_Evento.pdf: Certificado de participação em evento relacionado à área de tecnologia/educação, válido para comprovação de minha formação contínua.

Repositórios do Primeiro Semestre

Design_Profissional/: Trabalhos realizados no âmbito de design profissional, abordando as competências adquiridas e as práticas de design aplicadas aos projetos.

Pensamento_Computacional_e_Algoritmos/: Exercícios e projetos realizados em pensamento computacional e algoritmos, demonstrando a aplicação de lógica de programação e resolução de problemas.

Desenvolvimento_Front_End/: Projetos realizados no aprendizado de desenvolvimento front-end, incluindo HTML, CSS, JavaScript e frameworks utilizados no curso.

Modelagem_Banco_Dados/: Arquivos e projetos relacionados à modelagem de bancos de dados, abordando o uso de SQL, normalização e práticas de design de banco de dados.
